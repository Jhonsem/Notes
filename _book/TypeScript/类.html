类 Class

类 Class

公有，私有，受保护的修饰符：

public(默认)：

在TS中默认成员都是public

private(私有)：

当成员被标记为private时候，就不能在声明它的类的外部使用：

class Animal{ private name:string; constructor:(theName:string){ this.name=theName; } }

存取器 get,set:

TS支持通过getters/setters来截取对象成员的访问,类似于vue的compute计算返回值;

let passcode="test key";
